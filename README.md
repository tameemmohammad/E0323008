# Stage 1

Designed REST APIs for retrieving notifications and supporting real-time notification delivery.

# Stage 2

Selected PostgreSQL as the database and designed a notifications schema with indexing support.

# Stage 3

Improved query performance using indexes on notification type and timestamp columns.

# Stage 4

Enhanced scalability using pagination, caching, and optimized API response handling.

# Stage 5

Implemented a queue-based notification processing approach for reliable bulk notification delivery.

# Stage 6

Implemented a Priority Inbox that ranks notifications based on type priority (Placement > Result > Event) and recency, displaying the top 10 notifications.

# Stage 7

Developed a responsive React application with notification listing, priority inbox view, and API integration.
